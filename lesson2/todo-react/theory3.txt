
что такое компоненты в react и зачем они нужны?
в react все строится из компонентов
компонент - это обычная js функция, которая начинается с заглавной буквы и возвращает разметку(.jsx)
Переиспользуемый кусочек интерфейса

компонент: 
const Hello = () => <p>Hello!</p>


const App = (props - входные параметры.)  => {
  return (
    <>
        <Hello />
        <Hello />
        <h1 className="todo__title">To Do List</h1>
    </>
  )
}

export default App

как выделять компонент? смотря на верстку в фигме, можно заметить повторяющиеся элементы - это и есть компоненты. кнопки, карточки, поля ввода, иконки, навигационные панели и т.д.
