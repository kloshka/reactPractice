useState – это переменная, которая «помнит» своё значение между рендерами и при изменении вызывает перерисовку компонента.

useEffect – это действие, которое должно произойти в ответ на изменение состояния или после первого появления компонента на экране.

useEffect – код несвязанный на прямую с отрисовкой интерфейса:

Запросы на сервер (fetch / axios)

Подписки на события (WebSocket, resize)

Таймеры (setTimeout, setInterval)
Интеграция с библиотеками, не связанными с React

import { useEffect } from 'react'

const Example = () => {
    useEffect(() => {
    // ваш побочный эффект
}, [/* зависимости */]) 
1ый аргумент - функция, которая выполняет нужное действие
